# B-CareHub

住宅型有料老人ホーム + 訪問介護・訪問看護事業向けの業務管理システム。

利用者を中心に、食事発注（朝パン・ジュース、昼弁当、夕弁当、業者別）、日用品、固定費、介護・訪問看護利用料、立替金、書類、タスクを一元管理することを目的とする。

## このリポジトリの内容

- `要件定義書_B-CareHub.md` — システム要件定義書 v1.0（1,700行超）
- `app/` — Next.js App Router によるプロトタイプ画面
- `lib/data.ts` — モックデータ（DB置換前提のシェイプ）
- `components/` — 共通コンポーネント

## プロトタイプの画面

| パス | 画面 |
|---|---|
| `/` | 施設ダッシュボード |
| `/users` | 利用者一覧 |
| `/users/[id]` | 利用者詳細（基本情報＋当月請求明細） |
| `/meals` | 食事発注カレンダー（月間） |
| `/billing` | 月次請求管理 |
| `/goods` | 日用品管理 |
| `/documents` | 書類・タスク管理 |

## 開発

```bash
npm install
npm run dev
```

http://localhost:3000 にアクセス。

## デプロイ

このリポジトリは Vercel にプッシュすると自動的にビルド・デプロイされる構成です。

## 技術スタック

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS

## 注記

現在のプロトタイプは UI 検証用のモックです。本番実装時は要件定義書のとおり PostgreSQL + Prisma 等のデータ層に差し替えます。
